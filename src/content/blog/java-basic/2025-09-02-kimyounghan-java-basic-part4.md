> **출처**: [김영한의 자바 기본 강의](https://inf.run/2714a) - Section 4: 객체 지향 프로그래밍

객체 지향 프로그래밍은 데이터와 그 데이터에 대한 행동을 하나의 객체로 묶어서 프로그래밍하는 방법이다.

클래스는 멤버 변수와 메서드를 정의할 수 있다.

객체는 자신의 메서드를 통해 자신의 멤버 변수에 접근할 수 있다. 속성과 기능만 존재한다. 속성은 멤버 변수로, 기능은 메서드로 구현된다.

캡슐화는 속성과 기능을 하나로 묶어서 필요한 기능을 메서드를 통해 외부에 제공하는 것을 뜻한다.
